# Task Manager - Консольный диспетчер задач
Простой консольный диспетчер задач для мониторинга системы и управления процессами.
## Возможности
 Просмотр системной информации (CPU, память, диски)
 Отображение списка запущенных процессов
 Детальная информация о конкретном процессе
 Завершение процессов
 Мониторинг производительности системы
## Установка
1. Убедитесь, что у вас установлен Python 3.6+
2. Склонируйте или скачайте проект
3.В самом коде уже импортированы все нужные для работы библиотеки,так что дополнительно устанавливать ничего не нужно
